# กล้องวงจรปิด (CCTV)

กล้องวงจรปิดจัดเป็น *Physical Security Control*  
ที่มีลักษณะเป็น *Preventive Control (การควบคุมเชิงป้องกัน)*

---

## เหตุผลที่จัดเป็น Preventive Control

### 1.ยับยั้งพฤติกรรมที่ไม่พึงประสงค์ (Deterrence)
- การมีกล้องปรากฏชัดเจนทำให้ผู้ไม่หวังดีลังเลก่อนก่อเหตุ
- ลดโอกาสเกิดการโจรกรรม การทำลายทรัพย์สิน หรือการละเมิดข้อมูล
- ช่วยสร้างความตระหนักด้านความปลอดภัยให้พนักงานและผู้มาติดต่อ 

### 2. ป้องกันเหตุการณ์ก่อนเกิดความเสียหาย
- แม้กล้องจะบันทึกข้อมูล แต่การ “มีกล้อง” ช่วยลดความเสี่ยงตั้งแต่ต้นทาง
- ใช้ร่วมกับระบบแจ้งเตือนแบบ Real-time เพื่อตรวจจับพฤติกรรมผิดปกติ

### 3. ควบคุมและตรวจสอบการเข้าถึงพื้นที่
- ใช้ตรวจจับการเข้า–ออกในพื้นที่ควบคุม เช่น ห้องเก็บข้อมูล
- ป้องกันไม่ให้บุคคลที่ไม่ได้รับอนุญาตแอบเข้ามา

### 4. เพิ่มความปลอดภัยโดยรวมขององค์กร
- ช่วยตรวจสอบกิจกรรมโดยรอบอาคาร
- สนับสนุนการทำงานของทีมรักษาความปลอดภัย
- ลดความเสี่ยงของเหตุการณ์ด้านความปลอดภัยที่อาจเกิดขึ้นโดยไม่ตั้งใจ

### 5. รองรับการติดตามเหตุการณ์ย้อนหลัง (แต่ยังอยู่ในเชิงป้องกัน)
แม้ฟังก์ชันวิเคราะห์ย้อนหลังจะเข้าข่าย Detective Control แต่ ระบบกล้องในเชิงป้องกัน 
คือการติดตั้งเพื่อให้ “ผู้ก่อเหตุไม่กล้าลงมือ” จึงจัดเป็น Preventive Control ได้

- ลดความเสี่ยงก่อนเกิดเหตุ

- ทำให้องค์กรมีระบบป้องกันตั้งแต่ต้นทาง
